### Description:

Find all 10 letter long substrings that occur more than once for a given string.

 

### Example 1:

Input: s = "AAAAACCCCCAAAAACCCCCCAAAAAGGGTTT"

Output: ["AAAAACCCCC","CCCCCAAAAA"]

### Example 2:

Input: s = "AAAAAAAAAAAAA"

Output: ["AAAAAAAAAA"]
 


### Constraints:

- 0 <= s.length <= 10^5
- s[i] is 'A', 'C', 'G', or 'T'.
